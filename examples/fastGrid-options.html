1.options

名称 可选值 描述
frame true, false 设置width和height的值是否将边框计算在内。
width '100%', 500, '500px' 表格宽度。
height '100%', 500, '500px' 表格高度。
url false, string ajax请求数据的地址。
params {}, object, function(){ return {}; } ajax请求的参数。可以是普通对象或函数。 函数返回一个参数对象，每次调用ajax时调用。
method 'post', ‘get’ ajax提交方式。
items [], Array 数据使用本地对象数组。
nowrap false, true 表格显示的数据超出列宽时是否换行。
multiSelect false, true 数据使用本地对象数组。
loadingText ‘正在载入...’, string 数据载入时的提示文字。
noRecordText ‘没有数据’, string 没有数据时的提示文字。
cols [], Array 数据模型。
sortName '', string 排序的字段名。
sortStatus 'asc', string 排序方向。
remoteSort false, true 是否使用服务器端排序。当值为true时，sortName和sortStatus会作为参数放入ajax请求中。
autoLoad true, false 是否表格准备好时加载数据。
onSuccess function(){}, function 数据加载成功后执行。
onError function(){}, function 数据加载失败后执行。
onSelected function(item, rowIndex, colIndex){}, function 表格中的数据被选择时执行。item：当前数据行对应的对象；rowIndex：行索引；colIndex：列索引。

2.col

名称 可选值 描述
title string 标题文字。
width int 标题宽度像素值。
name string 字段名。
type 'int','float' 字段类型。在本地排序时用于转换文字。
align 'left', 'center', 'right' 文字位置。
hidden false, true 是否隐藏列。
sortable false, true 是否可排序。
renderer function(val,item,items,rowIndex, row){ return val;} 显示规则。参数: val 值, item 对象, items 对象数组, rowIndex 行索引, row 行element; 返回值: 可以是文本，html文本和jQuery对象。

2.operation option

名称 参数 描述
load {}, [] 如果配置过ajax选项，则这里是ajax参数，如果传入对象数组，则为加载本地数据。
selected 无参数 返回选择行的对象数组。
select int, function(item){ return true; }, 'all' 选中参数条件的行。int: 选中索引行; function: 过滤函数, item参数为行数据; 'all': 选中所有行。
deselect int, function(item){ return true; }, 'all' 取消选中参数条件的行。int: 取消选择索引行; function: 过滤函数, item参数为行数据; 'all': 取消选中所有行。
